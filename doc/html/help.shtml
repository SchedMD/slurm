Getting Help

Try the following steps if you are having problems with SLURM:

  1. See if the problem is addressed in the SLURM FAQ.
  2. For run-time problems, try running the command or daemons in verbose mode (-v option), and see if additional information helps you resolve the problem.
  3. Customers of HP and Linux NetworX should contact their support staff.
  4. Send a detailed description of the problem and logs to slurm-dev@lists.llnl.gov.
