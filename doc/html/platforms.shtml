Platforms

Operating Systems

  • AIX—Slurm support for AIX has been thoroughly tested, but we know of no AIX installations using Slurm after 2008. See IBM AIX User and Administrator Guide for more information.
  • Linux—Slurm has been thoroughly tested on most popular Linux distributions using i386, ia64, and x86_64 architectures.
  • OS X—Slurm has run OS X in the past, but the current OS X linker does not support Slurm plugins. (OS X dynamically linked objects can be called by the main program. They may not call functions in the main program or other dynamically linked objects, which are features required by Slurm.)
  • Solaris—Slurm support for Solaris (OpenSolaris build 119) was added in version 2.1.
  • Other—Slurm ports to other systems will be gratefully accepted.

Interconnects

  • BlueGene—Slurm support for IBM's BlueGene/L, BlueGene/P and BlueGene/Q systems has been thoroughly tested. See Blue Gene User and Administrator Guide for more information.
  • Cray XT and XE—Operates as a scheduler on top of Cray's ALPS/BASIL software. Supported added in Slurm version 2.3. Note that Cray's ALPS/BASIL software necessitates some changes in Slurm behavior. See Cray User and Administrator Guide for more information.
  • Ethernet—Ethernet requires no special support from Slurm and has been thoroughly tested.
  • IBM Infiniband/Torrent—Slurm supports IBM's NRT (Network Resource Table) interface and IBM's PE (Parallel Environment).
  • Infiniband—Infiniband support has been thoroughly tested.
  • Myrinet—Myrinet, MPICH-GM and MPICH-MX are supported.
  • Quadrics Elan—Slurm support for Quadrics Elan 3 and Elan 4 switches are available in all versions of Slurm and have been thoroughly tested.
  • Sun Constellation—Resource allocation has been optimized for the three-dimensional torus interconnect.
  • Other—Slurm ports to other systems will be gratefully accepted.
