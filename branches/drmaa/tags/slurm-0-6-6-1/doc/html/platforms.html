Platforms

Operating Systems

  • AIX—SLURM support for AIX is available in version 0.4.
  • Blue Gene—SLURM support for IBM's Blue Gene system is available in version 0.4.
  • Linux—SLURM has been thoroughly tested on Red Hat Linux using both IA32 and IA64. We anticipate SLURM functioning properly on other Linux distributions as well.
  • Other—SLURM ports to other systems will be gratefully accepted.

Interconnects

  • Ethernet—Ethernet requires no special support from SLURM and has been thoroughly tested.
  • IBM Federation—SLURM support for IBM's Federation Switch is available in version 0.5.
  • Infiniband—Infiniband support is available in version 0.6.
  • Myrinet—Myrinet and MPICH-GM are support is available in version 0.4
  • Quadrics Elan—SLURM support for Quadrics Elan 3 and Elan 4 switches are available in all versions of SLURM and have been thoroughly tested.
  • Other—SLURM ports to other systems will be gratefully accepted.

For information about this page, contact slurm-dev@lists.llnl.gov.
